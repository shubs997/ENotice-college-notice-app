# college-notice-application
Java EE project made using JSP and Servlets with Model View Controller (MVC) architecture.


IDE : Apache Netbeans 8.2 or higher.

Server: Apache Tomcat

Database: MySQL

Libraries: JDBC Connector and JSTL

## Technology Stack
* HTML5
* CSS3
* ES6
* BOOTSTRAP
* JAVA EE
* MYSQL
